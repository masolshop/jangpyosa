### 2️⃣ 월 60시간 기준 수정 (중증 2배 인정) ✅

**문제점:**
- `workHoursPerWeek >= 60` 조건으로 **주당 60시간**을 기준으로 체크
- 그러나 DB에는 `workHoursPerWeek = monthlyWorkHours / 4.33`으로 저장되어 15~20시간 정도
- 따라서 중증 직원이 **단 한 명도 2배 인정되지 않음**

**수정 내용:**
```typescript
// ❌ 기존 (잘못된 주당 시간 체크)
if (emp.severity === "SEVERE" && emp.workHoursPerWeek >= 60) {
  levyRecognizedCount = 2.0;
}

// ✅ 수정 (월간 시간으로 환산)
const monthlyHours = emp.workHoursPerWeek * 4.33; // 주당 → 월간 환산
if (emp.severity === "SEVERE" && monthlyHours >= 60) {
  levyRecognizedCount = 2.0;
}
```

**효과:**
- 월 60시간 이상 근무하는 중증 직원이 **정확히 2배로 인정**
- 부담금 계산 시 인정 인원 수가 증가하여 **부담금 감소**
