## 🔍 핵심 개념 정리

### 1️⃣ **초과인원** (미달/초과 컬럼 표시)
```
초과인원 = 인정수(중증×2) - 의무고용인원(floor)
```

**용도**: 부담금/장려금 구분
- **음수(-)**: 미달 → 부담금 납부
- **양수(+)**: 초과 → 부담금 면제

**5월 예시**:
- 인정수 = 중증4명×2 + 경증7명×1 = **15.0**
- 의무고용 = floor(399 × 3.1%) = **12**
- 초과인원 = 15.0 - 12 = **+3명** ✅

---

### 2️⃣ **장려금 대상** (실제 지급 여부)
```
장려금 대상 = 실제 장애인수(중증×1) - 지급기준인원(ceil)
```

**중요**: 중증 장애인도 **1배**로 계산!

**5월 예시**:
- 실제 장애인 = **11명** (중증×1)
- 지급기준 = ceil(399 × 3.1%) = **13명**
- 장려금 대상 = 11 - 13 = **-2명** (부족)
- 따라서 장려금 **0원** ✅

## 💡 왜 5월은 초과인데 장려금이 0원인가?

### **이중 기준 시스템**

1. **부담금/장려금 구분** (초과인원)
   - 중증 **2배수** 적용
   - **초과 +3명** → 부담금 없음 ✅

2. **장려금 지급 여부** (실제 인원)
   - 중증 **1배수** 적용
   - **부족 -2명** → 장려금 없음 ❌

### **고용부 규정 의도**
- 초과인원: 중증 장애인을 **우대**하여 2배 인정
- 장려금: 실제 고용한 **실인원**으로만 계산

따라서:
- **인정수(15.0) > 의무고용(12)** → 초과 상태 (부담금 면제)
- **실제인원(11) < 지급기준(13)** → 기준 미달 (장려금 없음)
