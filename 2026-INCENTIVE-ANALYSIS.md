## 📊 장애인고용개선장려금 상세 분석

### 1. 지원 대상 (매우 제한적!)

#### ✅ 지원 대상 사업주
```
✔ 상시근로자 수: 50인 이상 100인 미만
✔ 고용의무 미이행 사업주
✔ 민간기업 (국가/지자체/공공기관 제외)
✔ 2026년 1월 1일 이후 중증장애인 신규 채용
```

#### ❌ 제외 대상
- 상시근로자 50인 미만
- 상시근로자 100인 이상
- 고용의무 이행 중인 사업주
- 국가/지자체/공공기관

### 2. 지원 금액 (시행지침 V. 지원 금액 참조)

#### 📌 지급 단가

| 구분 | 지급 단가 | 조건 |
|------|----------|------|
| **중증남성** | **35만원/월** | 지급 단가와 월임금 60% 중 낮은 금액 |
| **중증여성** | **45만원/월** | 지급 단가와 월임금 60% 중 낮은 금액 |

**계산식**:
```
월 장려금 = MIN(지급 단가, 월임금액 × 60%)
```

**예시**:
```
중증남성, 월급 200만원:
→ MIN(350,000원, 2,000,000원 × 60%) = MIN(350,000원, 1,200,000원)
→ 350,000원 지급

중증남성, 월급 50만원:
→ MIN(350,000원, 500,000원 × 60%) = MIN(350,000원, 300,000원)
→ 300,000원 지급
```

### 3. 지원 방식

#### ⏱️ 지원 기간
- **최대 12개월** (해당 중증장애인 1인당)
- 증가한 월부터 지급 시작
- 익월부터 월별 신청 가능

#### 🔢 지원 인원
```
지원 인원 = 증가한 중증장애인 수
※ 중증장애인 2배수 산정 특례 미적용
```

**중요**: 
- 증가 판단 시 **2배수 적용 안 함**
- 의무고용 판단 시에만 2배수 적용

### 4. 중복지원 제한

#### ❌ 중복 불가
```
- 고용보험법에 따른 장려금/지원금
- 산업재해보상보험법에 따른 장려금
- 사회적기업 육성법에 따른 지원금
→ 차액만 지급
```

#### ✅ 중복 가능 (전액 지급)
```
- 고용유지지원금
- 청년내일채움공제
```
