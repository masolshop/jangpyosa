## ❌ 가능한 에러 케이스

### Case 1: 하위 조직이 있는 경우
**에러 메시지:**
```
하위 조직이 있어 삭제할 수 없습니다. 먼저 하위 조직을 재배치하거나 삭제해주세요.
```

**해결 방법:**
- 하위 직원을 먼저 다른 곳으로 이동
- 또는 하위 직원을 먼저 삭제

**확인 방법:**
- 본부장: 소속 지사가 있는지 확인
- 지사장: 소속 매니저가 있는지 확인

### Case 2: 인증 문제
**에러:** `401 Unauthorized` 또는 `403 Forbidden`

**해결 방법:**
1. 다시 로그인
2. localStorage에 `accessToken`이 있는지 확인:
   ```javascript
   // 브라우저 콘솔에서 실행
   localStorage.getItem('accessToken')
   ```

### Case 3: 네트워크 에러
**에러:** 요청이 아예 전송되지 않음

**확인:**
- 인터넷 연결 확인
- 서버 상태 확인: https://jangpyosa.com/api/health

### Case 4: 서버 내부 에러
**에러:** `500 Internal Server Error`

**확인:**
- 서버 로그 확인 필요
- PM2 로그에서 에러 스택 확인
