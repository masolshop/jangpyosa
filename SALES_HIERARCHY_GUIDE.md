## 🔧 관리 기능

### 본부장/지사장 공통 기능

#### 1️⃣ 매니저 생성
```
대시보드 → ➕ 매니저 생성 버튼

입력 항목:
- 이름 (필수)
- 전화번호 (필수, 예: 01012345678)
- 이메일 (선택)
- 비밀번호 (필수, 최소 6자)

결과:
- User 생성 (role: EMPLOYEE)
- SalesPerson 생성 (role: MANAGER, managerId: 본인)
- isActive: true (즉시 활성화)
- 생성된 계정으로 즉시 로그인 가능
```

#### 2️⃣ 정보 수정
```
대시보드 → 하위 조직 카드 → ✏️ 수정 버튼

수정 가능 항목:
- 이름
- 전화번호
- 이메일

주의:
- 비밀번호는 수정 불가 (슈퍼어드민만 가능)
```

#### 3️⃣ 활성화/비활성화
```
대시보드 → 하위 조직 카드 → 🔒 비활성화 / 🔓 활성화 버튼

효과:
- isActive 토글
- 비활성화 시 로그인 불가
- 활성화 시 로그인 가능
```

#### 4️⃣ 삭제
```
대시보드 → 하위 조직 카드 → 🗑️ 삭제 버튼

조건:
- 하위 조직이 없어야 삭제 가능
- 하위 조직이 있으면 에러 메시지

효과:
- SalesPerson 레코드 삭제
- 관련 활동 로그 기록
```

### 시나리오 3: 일반 매니저 회원가입

1. **회원가입**
   - URL: https://jangpyosa.com/admin/sales
   - 회원가입 탭 클릭

2. **본인 인증**
   - 주민번호 앞 6자리 입력
   - 주민번호 뒤 7자리 입력
   - 인증 성공

3. **정보 입력**
   - 전화번호: 01077778888
   - 이메일: manager@example.com
   - 비밀번호: password123
   - 본부 선택: 페마연 (본부장)
   - 지사 선택: (선택사항)

4. **승인 대기**
   - 가입 완료 메시지
   - isActive: false (비활성 상태)
   - 슈퍼어드민 승인 필요

5. **슈퍼어드민 승인**
   - 슈퍼어드민 로그인
   - 영업 관리 페이지 → 매니저 찾기
   - 활성화 버튼 클릭
   - isActive: true로 변경

6. **로그인**
   - ID: 01077778888, PW: password123
   - 로그인 성공 → 대시보드 페이지

## 🎯 정리

### 계층별 권한 요약

| 역할 | 지사 생성 | 매니저 생성 | 수정 | 삭제 | 활성화 | 대시보드 |
|-----|---------|-----------|-----|-----|--------|---------|
| 슈퍼어드민 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ 전체 |
| 본부장 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ 하위 |
| 지사장 | ❌ | ✅ | ✅ | ✅ | ✅ | ✅ 하위 |
| 매니저 | ❌ | ❌ | ❌ | ❌ | ❌ | ✅ 자신 |

### 비밀번호 규칙

- 본부/지사: 생성 시 입력한 비밀번호 (권장: 전화번호와 동일)
- 매니저: 회원가입 시 입력한 비밀번호 (최소 6자)
- 모든 계정: 전화번호 형식 상관없이 로그인 가능 (01012345678, 1012345678, 010-1234-5678)

### 계정 활성화 규칙

- 본부/지사: 생성 시 즉시 활성화 (isActive: true)
- 매니저: 회원가입 후 슈퍼어드민 승인 필요 (isActive: false → true)
