# 변경 사항 - 2026-02-18

## 🐛 대시보드 금액 0원 문제 해결

### 문제
- 데모 계정 "주식회사 페마연" (전화: 010-5555-6666) 로그인 시 대시보드에 금액이 0원으로 표시됨

### 원인
- `BuyerProfile.yearlyEmployeesJson` 필드가 `null`로 되어 있어 상시근로자 수 데이터가 없음
- Dashboard API는 이 데이터를 기반으로 부담금·장려금을 계산하므로 데이터 없으면 계산 불가

### 해결
데모 계정에 2026년 상시근로자 수 데이터 추가:

```json
{
  "2026": {
    "1": 120, "2": 120, "3": 125, "4": 125,
    "5": 130, "6": 130, "7": 135, "8": 135,
    "9": 140, "10": 140, "11": 145, "12": 145
  }
}
```

### 결과
- **총 부담금**: 22,698,000원
- **예상 감면**: 11,349,000원
- **순 부담금**: 11,349,000원

### 데이터베이스 변경

**회사 정보**:
- 회사명: 주식회사 페마연
- 회사 타입: BUYER (고용의무기업)
- 등록된 장애인 근로자: 2명
  - 홍길동 (중증, 주 80시간, 월급 2,060,740원)
  - 김영희 (중증, 주 40시간, 월급 2,500,000원)

**계산 로직**:
1. 월별 상시근로자 수: 120~145명
2. 의무고용인원 (3.1%): 4~5명
3. 인정 장애인 수: 3명 (홍길동 2명 인정 + 김영희 1명 인정)
4. 부족인원: 1~2명
5. 월별 부담금: 부족인원 × 1,260,000원
6. 연간 총 부담금: 22,698,000원

### 테스트
```bash
# API 테스트
curl http://localhost:4000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"010-5555-6666","password":"test1234","userType":"BUYER"}'

# 받은 accessToken으로 대시보드 조회
curl http://localhost:4000/dashboard?year=2026 \
  -H "Authorization: Bearer <accessToken>"
```
