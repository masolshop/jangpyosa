## 🔐 로그인 방법

### 기업 관리자 로그인
1. **URL**: https://jangpyosa.com/login
2. **아이디**: 영문+숫자 (예: `pema_admin`)
3. **비밀번호**: `test1234`
4. **권한**: 
   - 직원 관리 (등록/수정/삭제)
   - 연차 승인/거부
   - 휴가 신청 승인/거부
   - 출결 관리
   - 통계 조회

### 장애인 직원 로그인
1. **URL**: https://jangpyosa.com/employee/login
2. **휴대폰번호**: 010-XXXX-XXXX (실제 핸드폰 번호)
3. **비밀번호**: `test1234`
4. **권한**:
   - 본인 연차 조회
   - 휴가 신청
   - 출결 기록
   - 업무 지시 조회

## 📊 데이터 확인 방법

### 실제 장애인 직원 핸드폰 번호 확인
```bash
# SSH로 서버 접속
ssh -i ~/.ssh/jangpyosa ubuntu@jangpyosa.com

# 데이터베이스 직접 확인
cd /home/ubuntu/jangpyosa/apps/api
npx prisma studio
```

또는 SQL 쿼리:
```sql
-- 페마연구소 직원 조회
SELECT name, phone FROM DisabledEmployee 
WHERE buyerId = (SELECT id FROM BuyerProfile WHERE companyId = 
  (SELECT id FROM Company WHERE name = '페마연구소'));

-- 공공기관A 직원 조회
SELECT name, phone FROM DisabledEmployee 
WHERE buyerId = (SELECT id FROM BuyerProfile WHERE companyId = 
  (SELECT id FROM Company WHERE name = '공공기관A'));

-- 행복한표준사업장 직원 조회
SELECT name, phone FROM DisabledEmployee 
WHERE buyerId = (SELECT id FROM BuyerProfile WHERE companyId = 
  (SELECT id FROM Company WHERE name = '행복한표준사업장'));
```

## ⚠️ 주의사항

### 핸드폰 번호 규칙
장애인 직원 핸드폰 번호는 **고정된 패턴**으로 생성됩니다:
- **페마연구소**: 010-1001-0001 ~ 010-1001-0015 (15명)
- **공공기관A**: 010-2001-0001 ~ 010-2001-0012 (12명)
- **행복한표준사업장**: 010-3001-0001 ~ 010-3001-0015 (15명)

모든 장애인 직원의 비밀번호는 `test1234`로 동일합니다.

### 데이터 재생성 (필요시)
AWS 서버에서 데이터베이스를 초기화하고 시드 데이터를 재생성하려면:
```bash
ssh -i ~/.ssh/jangpyosa ubuntu@jangpyosa.com
cd /home/ubuntu/jangpyosa/apps/api
npm run db:push -- --force-reset --accept-data-loss
npm run db:seed
pm2 restart all
```

## 🎯 테스트 시나리오

### 시나리오 1: 기업 관리자 → 직원 관리
1. `pema_admin`으로 로그인
2. 대시보드 → 직원 관리
3. 15명의 직원 목록 확인
4. 연차 현황 탭에서 각 직원별 연차 잔여 확인

### 시나리오 2: 장애인 직원 → 휴가 신청
1. 장애인 직원 계정으로 로그인 (핸드폰번호 + test1234)
2. 내 연차 조회
3. 휴가 신청
4. 승인 대기

### 시나리오 3: 관리자 → 휴가 승인
1. `pema_admin`으로 로그인
2. 휴가 관리 → 신청 목록
3. 대기 중인 신청 승인/거부

## 🔗 관련 API 엔드포인트

### 기업 관리자용
- `GET /api/companies/my` - 내 회사 정보
- `GET /api/calculators/company/:id/employees` - 회사 직원 목록
- `GET /api/annual-leave/company/:companyId` - 전체 연차 현황
- `GET /api/leave/requests` - 휴가 신청 목록
- `PATCH /api/leave/requests/:id/approve` - 휴가 승인
- `PATCH /api/leave/requests/:id/reject` - 휴가 거부

### 장애인 직원용
- `GET /api/annual-leave/employee/:employeeId` - 내 연차 조회
- `GET /api/leave/requests/my` - 내 휴가 신청 내역
- `POST /api/leave/requests` - 휴가 신청
